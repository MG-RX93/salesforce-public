## Create Scratch Orgs

- Create Scratch Org
```bash
sf org create scratch --definition-file config/project-scratch-def.json --alias <scratch_org_name> --set-default --target-dev-hub <dev_hub_name>
```

- Open Scratch Org
```bash
sf org open --target-org <scratch_org_name>
```
